# 豆瓣电影助手

让豆瓣电影支持下载bt种子。

[Chrome webstore 安装地址](https://chrome.google.com/webstore/detail/%E8%B1%86%E7%93%A3%E7%94%B5%E5%BD%B1%E5%8A%A9%E6%89%8B/kolgihmifdjohkfddelnhcmaegckgpkh)

## Features

- 支持一键云播 (需要迅雷会员)
    - `Shift+单击` 云播种子或地址
    - `Ctrl+单击` 添加种子或地址到云播空间
    - `Alt+单击` 简洁云播页面
- 支持绝大部分电影多个版本的bt种子或地址下载
- 可切换不同平台的搜索结果
- 支持字幕下载
